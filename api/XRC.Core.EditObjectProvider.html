    Class EditObjectProvider

    EditObjectProvider is a MonoBehaviour that provides functionality to select and interact with a target object.

    Inheritance
    Object
    EditObjectProvider
    Namespace: XRC.Core
    Assembly: cs.temp.dll.dll
    Syntax
    public class EditObjectProvider : MonoBehaviour
    Remarks

    This component is used in edit tools that implement the IEditTool interface, such as XRC Mesh Tool, XRC Scale Tool, and XRC Color Tool, for providing the game object to be edited.

    Methods

    StartRun()

    Start the edit tool and provide the edit object based on the recently selected interactable. If the SnapBack property is set to true, the object will be moved back to its grabbed position and orientation. If the StartEditOnSet property is set to true, the edit tool will be started immediately after the object has been set.

    Declaration

    StopRun()

    Remove the edit object from the edit tool and enables the interactable so it can be selected again. Stops the run on the edit tool.
